# 制作物１
矢野翔大の制作物のソースコードです。
こちらの制作物は現在研究室のチームで進めている研究で開発した、「日本語を勉強している外国人に対して、日本語助詞の学習を支援する」ために開発しています。
現時点ではまだ研究途中、未公開のアプリケーションですので内部的な情報等が入っている多くのソースコードは掲載しておりません。ご了承ください。

以下、ソースコードの簡単な説明になります。

## GameView
このViewは提示された問題文に対して、ドラッグ形式で穴埋めの選択肢を選ぶというゲーム形式のビューを形成します

## HintHandView
GameViewでのドラッグ操作に視覚的ヒントを与えるために指を下向きに動かすアニメーションを形成しています

## RubyLine
アプリケーション内で表示される振り仮名付きの文字列を形成しています
